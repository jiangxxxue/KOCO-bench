# ToolBrain智能工具学习框架

## 1. 项目概述

ToolBrain是一个面向工具使用能力学习的轻量级框架，围绕smolagents工具生态构建，提供统一的工具描述与校验、跨框架适配执行、以及面向多种学习策略的可训练数据流。该系统通过集成多种学习算法和适配器模式，为开发者提供智能化的工具使用能力训练和推理功能。ToolBrain采用模块化设计，结合动态工具卡片生成和丰富的适配器生态系统，实现了高效的工具使用能力学习和应用。

## 2. 核心功能

### 工具描述与提示构造
- 从工具对象抽取统一元数据与参数模式，输出标准化工具卡片
- 训练样本生成提示拼装与最少工具调用约束
- 批量工具卡片渲染和训练示例生成提示构建

### 工具与模型校验
- 对工具列表进行严格类型、可调用性、重名校验
- 对外部模型进行可调用生成接口签名检查
- 工具集合验证和模型接口检查

### 跨框架适配与执行追踪
- 统一适配器接口，兼容原生代理与LangChain代理
- 支持聊天模型与工具调用包装，产出可训练轨迹
- 基础适配器抽象和工具调用包装

### 学习策略与训练优化
- 多种学习算法支持，包括DPO、GRPO和监督学习
- 低秩适应微调，降低显存开销
- 高效的模型训练和推理能力

## 3. 技术特点

### 基于smolagents工具生态
- 与工具和模型强关联，直接消费与验证工具定义
- 通过统一的工具卡片确保训练与推理阶段工具约束一致性
- 支持多种工具类型和模型格式

### 可训练数据流与最小改动接入
- 通过适配器产出高保真轨迹，包含思考、工具代码、工具输出和最终答案
- 无侵入接入多种代理，保留原生调用范式
- 支持多种训练数据格式和轨迹记录方式

### 轻量高效的训练优化路径
- 支持低秩适应细化，降低显存开销
- 提示构造与工具约束面向数据合成与策略学习直接使用
- 多种学习算法支持，包括直接偏好优化和群体相对策略优化

### 丰富的适配器生态系统
- 基础适配器抽象，提供统一接口定义
- 原生代理适配器，兼容smolagents框架
- LangChain适配器，支持LangChain生态集成
