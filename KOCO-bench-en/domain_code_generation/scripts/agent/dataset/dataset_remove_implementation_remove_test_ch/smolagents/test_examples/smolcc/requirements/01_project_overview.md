# SmolCC智能代码助手

## 1. 项目概述

SmolCC (Small Code Companion) 是一个基于smolagents框架构建的智能代码助手系统。该系统通过集成多种代码操作工具，为开发者提供智能化的代码编辑、文件管理和系统操作能力。SmolCC采用ReAct (Reasoning and Acting) 框架，结合动态系统提示和丰富的工具生态系统，实现了高效的代码理解和操作能力。

## 2. 核心功能

### 代码编辑与查看
- 文件内容查看与语法高亮显示
- 代码编辑和保存操作
- 基于正则表达式的代码搜索
- 多文件内容替换和模式替换

### 文件系统管理
- 目录结构浏览和文件列表查看
- 通配符模式的文件查找
- 文件操作和目录导航

### 系统命令执行
- Bash命令执行和交互式Shell会话
- 环境感知和状态管理
- 安全命令执行机制

## 3. 技术特点

### 基于smolagents框架
- 继承ToolCallingAgent实现工具调用能力
- 集成ReAct推理框架
- 支持多工具协同工作

### 动态提示系统
- 上下文感知的系统提示生成
- Git仓库状态自动检测
- 实时工具文档集成

### 丰富的工具生态系统
- 代码编辑工具: ViewTool, EditTool, ReplaceTool
- 文件管理工具: LsTool, GlobTool
- 代码搜索工具: GrepTool
- 系统命令工具: BashTool
